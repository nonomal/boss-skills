# boss.skill

Distill a boss into an AI Skill.

This skill is designed to learn three things from imported chats, meeting notes, review comments, and project documents:

- how the boss evaluates projects and plans
- how the boss speaks, pushes, criticizes, and asks questions
- how you should manage upward with this specific boss

The generated skill has three parts:

- `Boss Judgment`
- `Managing Up`
- `Persona`

Core command:

```text
/create-boss
```

Generated commands:

- `/{slug}`
- `/{slug}-judgment`
- `/{slug}-management`
- `/{slug}-persona`

Typical use cases:

- simulate how your boss will review a proposal
- practice how to report bad news or project risk
- prepare a better update before a 1:1 or weekly sync
- learn how to ask for resources or priority with this specific boss
